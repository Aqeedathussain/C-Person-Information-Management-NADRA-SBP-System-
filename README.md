# C++ Person Information Management System

A multi-level inheritance C++ program that manages personal, banking, and identification details of multiple users.  
The project demonstrates OOP concepts such as inheritance, encapsulation, and data retrieval via CNIC search.

---

## Features
- Multi-level inheritance: `NADRA` → `SBP` → `Person`  
- Input and display of personal information, CNIC, and address  
- Input and display of bank account details  
- Job and filer/non-filer status tracking  
- CNIC-based search to quickly retrieve user information  

---

## Concepts Used
- Classes & Objects  
- Multi-level Inheritance  
- Encapsulation of private members  
- Array of objects  
- Input validation and search functionality  

---
